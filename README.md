# Sales Analysis Dashboard — Excel & Pivot Tables

An interactive sales analytics dashboard built entirely in Microsoft Excel 
using Pivot Tables, Power Query, and dynamic slicers — no code required.

## Dataset
- 9,994 order records across 4 years (2014–2017)
- 5,009 unique orders | 793 customers | 49 U.S. states
- 3 product categories: Furniture, Office Supplies, Technology
- 3 customer segments: Consumer, Corporate, Home Office
- 4 shipping modes | 4 regions (East, West, Central, South)

## Dashboard Features
- KPI cards: Total Sales ($2.3M), Total Profit, Margin %, Orders, Returns
- Sales trend analysis by month and year
- Profit & sales breakdown by category and sub-category
- Customer segment performance comparison
- Regional and state-level sales distribution
- Shipping mode analysis
- Dynamic slicers for filtering by region, year, and month

## Tools & Techniques
- Microsoft Excel (Pivot Tables, Pivot Charts)
- Power Query (data cleaning and transformation)
- Dynamic slicers for interactive filtering
- KPI cards with conditional formatting
- Custom calculated fields for margin %

## Skills Demonstrated
- Data cleaning and preparation
- Business KPI definition and tracking
- Visual storytelling with data
- Dashboard design for non-technical stakeholders
